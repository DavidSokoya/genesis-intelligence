# Genesis Intelligence Website Redesign

Redesign and optimization of the Genesis Intelligence corporate website. The primary goal was to modernize the user interface, significantly improve performance and accessibility, and create a more engaging and intuitive user experience.

- **Original Website:** [https://genesisintelligence.org/](https://genesisintelligence.org/)  
- **Redesigned Website:** [https://genesisintel.netlify.app/](https://genesisintel.netlify.app/)  
- **GitHub Repository:** [https://github.com/DavidSokoya/genesis-intelligence](https://github.com/DavidSokoya/genesis-intelligence)

## Project Overview

The original website suffered from slow load times and had an outdated design that did not align with the company's professional image. This project involved a complete front-end overhaul using modern technologies to address these issues and deliver a high-performance, visually appealing, and user-friendly website.

### Key Features & Improvements

- **Modern UI/UX:** Implemented a clean, professional, and fully responsive design using a modern aesthetic that enhances readability and user engagement.  
- **Performance Optimization:** Page load times were drastically reduced   
- **Improved Information Architecture:** Re-structured the site's navigation and content layout to ensure visitors can find information easily and intuitively.  


### Tech Stack

- **Language:** JavaScript  
- **Styling:** Tailwind CSS  
- **Deployment:** Netlify  

## Getting Started

To get a local copy up and running, follow these steps:

1. Make sure you have Node.js and npm installed.
2. Clone the repository:  
```bash
   git clone https://github.com/DavidSokoya/genesis-intelligence.git
````

3. Navigate to the project directory:

```bash
   cd genesis-intelligence
```
4. Install NPM packages:

```bash
   npm install
 ```
5. Run the development server:

```bash
   npm run dev
```
6. Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.
